My Projects

ePortfolio Generator (Stand-alone Java app)

Github
Through usage of this java application users can create their own ePortfolios (webpages) by interfacing with my intuitive user interface. Users may choose between 5 google fonts, 5 layouts, and 5 color schemes for each page of their ePortfolio and add images, paragraphs, headers, lists, videos, as well as slideshows to any of their pages and then export their project as an HTML5 application. This app saves and loads projects through use of JSON files and through the same methods uploads user-generated content to premade HTML documents.

Technologies used:

HTML5&CSS3, Javascript, JQuery, JSON, Java, JavaFX.

Abnormal Psychologies (webpage)

link
This is a webpage I developed for a Special Topics Computer Science course at Stony Brook University on Mobile Application and User Interface Development. It's a database of psychological disorders which lets users post their own "stories" about living with a particular disorder as well as read the posts of other users. I make ajax calls to google spreadsheets in order to access these posts.

Technologies used:

Photoshop, HTML5&CSS3, Javascript, JQuery, Ajax, Google Spreadsheets as a database for storing and retrieving user posts.
